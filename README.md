# Olivia Harper Homes - Portfolio Clone

A high-fidelity technical clone of the Olivia Harper Homes website, built to demonstrate modern frontend engineering capabilities.

## Features

- **Pixel-Perfect UI:** Exact replication of the original site's typography, colors, and layout.
- **Advanced Animations:** Implemented using Framer Motion and Lenis for smooth, scroll-driven interactions.
- **Glassmorphism:** Custom modular glass UI design for navigation and interface elements.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
- **Modern Tech Stack:** Built with Next.js 16 (App Router), React 19, and Tailwind CSS v4.

## Technical Implementation

- **Framework:** Next.js 16
- **Styling:** Tailwind CSS v4
- **Motion:** Framer Motion & Lenis (Smooth Scroll)
- **Icons:** Lucide React
- **Typography:** Local Gallient Serif & Inter Sans

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
