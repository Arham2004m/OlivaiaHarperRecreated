# Tech Stack Analysis - oliviaharperhomes.com

## Original Site
- **Design Agency:** Bangluxor.
- **Framework/CMS:** Likely Webflow or custom WordPress.
- **Styling:** Custom CSS / Utility-based.
- **Icons:** Custom SVGs or high-end icon library.
- **Animations:** GSAP or CSS Transitions.

## Next.js Implementation (Our Goal)
- **Framework:** Next.js 16 (App Router, React 19).
- **Language:** TypeScript (Strict).
- **Styling:** Tailwind CSS v4.
- **UI Components:** shadcn/ui (Radix Primitives).
- **Icons:** Lucide React + Extracted SVGs.
- **Typography:** Google Fonts (Playfair Display, Inter).
- **Animations:** Framer Motion (for smooth transitions and entrance animations).
- **Assets:** Optimized images via `next/image`.
