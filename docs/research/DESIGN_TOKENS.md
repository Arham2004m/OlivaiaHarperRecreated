# Design Tokens - oliviaharperhomes.com

## Colors

| Category | Value | Description |
|----------|-------|-------------|
| Background | `#FFFFFF` | Primary background, gallery-like feel |
| Off-white | `#F9F9F9` | Secondary background sections (inferred) |
| Primary Text | `#1A1A1A` | Headings and primary body text |
| Secondary Text | `#4A4A4A` | Muted descriptions and metadata (inferred) |
| Accent | Slate/Bronze | Muted sophisticated tones for CTA and highlights |
| Border | `#E5E5E5` | Subtle dividers and card borders |

## Typography

### Font Families
- **Serif:** *Playfair Display* (or similar high-contrast serif) - Used for primary headings.
- **Sans-Serif:** *Inter* / *Montserrat* (or similar clean sans-serif) - Used for body and UI.

### Scale
- **Hero Title:** ~80px-120px, Bold
- **Section Headings:** ~48px-64px, Semi-bold
- **Sub-headings:** ~24px-32px, Medium
- **Body:** 16px-18px, Regular
- **Small/Metadata:** 12px-14px, Regular/Light

## Spacing
- **Section Padding:** 100px - 150px
- **Grid Gap:** 20px - 40px
- **Line Height:** ~1.6 for body text

## Radius
- Minimal to no border radius on large images/sections (sharp architectural look)
- Small radius (~4px) for buttons (if any)
